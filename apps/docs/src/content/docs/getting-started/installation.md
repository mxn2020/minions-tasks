---
title: Installation
description: Install Minions Tasks packages
---

## TypeScript / Node.js

```bash
npm install @minions-tasks/sdk minions-sdk
```

## Python

```bash
pip install minions-tasks
```

## CLI

```bash
npm install -g @minions-tasks/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
