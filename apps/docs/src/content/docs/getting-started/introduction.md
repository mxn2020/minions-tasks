# Minions Tasks

Task and work management across agents, humans, and workflows

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-tasks/sdk`) — Core library for Node.js
- **Python SDK** (`minions-tasks`) — Python bindings
- **CLI Tool** (`@minions-tasks/cli`) — Command-line interface
- **Web App** — Interactive playground at [tasks.minions.wtf](https://tasks.minions.wtf)
